# Groq API Documentation

## Authentication
All API requests require authentication using your API key. Include it in the Authorization header:
Authorization: Bearer YOUR_API_KEY

## Rate Limits
- Free tier: 50 requests/minute
- Pro tier: 500 requests/minute
- Enterprise: Custom limits

## Chat Completions Endpoint
POST https://api.groq.com/v1/chat/completions
Parameters:
- model: Required (e.g., "llama-4-scout-17b-16e-instruct")
- messages: List of message objects
- temperature: 0.0 to 1.0
- max_tokens: 1 to 4096
- top_p: 0.0 to 1.0
- stream: true/false

## Audio Endpoints
### Speech-to-Text
POST https://api.groq.com/v1/audio/transcriptions
Supported formats: WAV, MP3, M4A, FLAC
Models: whisper-large-v3

### Text-to-Speech
POST https://api.groq.com/v1/audio/speech
Parameters:
- model: playai-tts
- voice: Aaliyah-PlayAI, James-PlayAI, Maya-PlayAI
- input: Text to synthesize
- response_format: wav, mp3, opus

## Response Formats
- JSON (default)
- Text-only
- Verbose JSON (metadata included)
- Streamed chunks

## Models Available
### Language Models
- llama-4-scout-17b-16e-instruct
- mixtral-8x7b-32768
- gemma-7b-it

### Audio Models
- playai-tts (text-to-speech)
- whisper-large-v3 (speech-to-text)

## SDK Installation
Python:
pip install groq

JavaScript:
npm install groq-sdk

## API Root URL
All endpoints: https://api.groq.com/v1/

## Headers
Required:
Content-Type: application/json
Authorization: Bearer YOUR_API_KEY

Optional:
Groq-Organization: org_... (for enterprise)
